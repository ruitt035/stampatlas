# Stamp Atlas（旅游盖章收集）

## 运行
双击运行 [serve.bat](file:///c:/Users/tt/Desktop/coding/serve.bat)，然后在浏览器打开：

- http://127.0.0.1:5173/

## 使用
- 在地图上点击任意位置：添加标记并自动创建记录
- 点击标记或列表项：打开记录抽屉，填写文字并上传盖章图片
- 进入「长图」：生成并下载可分享的拼图长图
- 进入「设置」：导出/导入 JSON 备份、清空数据

## 说明
- 数据仅存储在本机浏览器（IndexedDB），不会上传服务器
- 若定位不可用：通常与浏览器权限或 HTTPS 限制有关，可忽略
